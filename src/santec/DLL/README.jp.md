# DLLについて

[![en](https://img.shields.io/badge/lang-en-blue.svg)](https://github.com/santec-corporation/Python-IL-STS/blob/main/src/santec/DLL/README.md)

## 概要
Santec DLLは、機器との接続を確立し、IL操作を実行するための基本的な基盤として機能します。
